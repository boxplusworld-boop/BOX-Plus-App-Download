# BOX+ App

BOX+ 就像一座私人视觉图书馆。

收藏来自世界各地的优秀作品，看过的、未读的、反复翻阅的，始终都安静地陈列在书架之上。

当你有空时，随手取下一本，由心出发。

BOX+ 是一款专为视觉审美资源打造的免费本地阅读与管理应用。

通过统一的资源结构，将原画、插画、摄影、影像、品牌、UI、3D 等设计作品整理为可离线浏览的沉浸式视觉资源库。
